Getting Started with TOML Toolbox

The TOML Toolbox provides functions for reading and writing TOML (Tom's Obvious Minimal Language) configuration files in MATLAB. This toolbox makes it easy to work with TOML files using native MATLAB structs.

What is TOML?

TOML is a minimal configuration file format that's easy to read due to obvious semantics. TOML is designed to map unambiguously to a hash table and is commonly used for configuration files in many programming languages and tools.

Installation

The TOML Toolbox is automatically added to your MATLAB path when you install it. You can verify the installation by typing:

which readtoml

Quick Start

Reading TOML Files

Use readtoml to read a TOML file into a MATLAB struct:

% Read a TOML configuration file
config = readtoml("config.toml");

% Access nested values using struct syntax
serverAddress = config.database.server;
port = config.database.port;

Writing TOML Files

Use writetoml to write a MATLAB struct to a TOML file:

% Create configuration as a struct
config.app_name = "My Application";
config.version = 2.1;
config.database.server = "localhost";
config.database.port = 5432;
config.features = ["auth", "logging", "metrics"];

% Write to TOML file
writetoml("config.toml", config);

Key Features

  • Round-trip support: Preserve key ordering when reading and writing files
  • Nested tables: Support for deeply nested configuration structures
  • Type safety: Automatic conversion between TOML and MATLAB types
  • File protection: Prevents accidental overwrites unless explicitly allowed
Note: This toolbox requires MATLAB R2022b or later due to the use of dictionary support for metadata tracking.
