TOML Toolbox Functions

The TOML Toolbox provides two main functions for working with TOML configuration files:

  • readtoml

    data = readtoml(filename)

    Read TOML configuration file into MATLAB struct. Supports optional metadata output for round-trip key ordering preservation.

  • writetoml

    writetoml(filename, data)

    Write MATLAB struct to TOML configuration file. Includes file overwrite protection and optional metadata input for preserving key order.
